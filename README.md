- 🧭 User-friendly interface for managing network settings

## Installation

```bash
pnpm i nmanager -g
```

## Usage

ℹ️ Show available commands and options:
```bash
nmanager
nmanager --help
```

🧭 List all commands and subcommands:
```bash
nmanager --commands
```
